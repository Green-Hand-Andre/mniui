## API

### Button Props

| 名称| 类型 | 默认值| 说明 | 可选值 |
| ----- | ---------------- | --------- | -------- | ----------- |
| color | string | —— | 按钮颜色 | 例：(255,255,255,1) |
| size | string | —— | 按钮大小 | big \| mo | size |
| type | string | —— | 按钮类型 | display |
